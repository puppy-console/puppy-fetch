# Prerequisites:

---

- ruby >= 3.2
- bundle >= 2.4.1
- git

---

# Installation

1. Add to Gemfile:
```rb
gem "puppy-fetch", git: "https://github.com/puppy-tools/puppy-fetch.git"
```

2. Bundle Install:
```sh
bundle install
```

# Usage

- As CLI:
```
bundle exec puppy-fetch 
```

- As Library:
```rb
require 'puppy-fetch'
```

# Prerequisites:

---

- ruby >= 3.2
- git

---

# Installation

---

1. Add to Gemfile:
```
gem "puppy-fetch", git: "https://github.com/puppy-tools/puppy-fetch.git"
```

2. Bundle Install:
```
bundle install
```

# Usage

3. Using as CLI:
```
bundle exec puppy-fetch GITHUB_OWNER GITHUB_REPO [options]
```

4. Using as Library:
```
require 'puppy-fetch'
```
